# Node todolist kata

#### 使用套件

- dotenv 16.4.5
- uuid 9.0.1
- eslint 8.57.0,
- eslint-config-standard 17.1.0,
- eslint-plugin-import 2.29.1,
- eslint-plugin-n 15.7.0,
- eslint-plugin-promise 6.1.1

#### 安裝套件

node 版本 18.x

```sh
npm install
```

#### 執行專案

```sh
npm run start
```

將 .env.example 更名為 .env
更改 PORT=XXXX

```sh
PORT=8080
```
